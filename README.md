# 🚀 AI-Powered Recruitment Platform

A modern recruitment platform that replaces traditional resume uploads with AI-assisted structured profile creation.

---

## 🎯 Objective

This platform eliminates the need for PDF resumes by using:
- AI-assisted input
- Structured data capture
- Smart profile building

---

## 🧠 Problem

Traditional hiring platforms suffer from:
- Inconsistent resume formats
- Poor parsing (ATS issues)
- Bias in evaluation
- Time-consuming screening

---

## 💡 Solution

We designed a system where:
- Candidates build structured profiles instead of uploading resumes
- AI assists in generating summaries and improving inputs
- Recruiters can easily compare candidates

## ⚡ Setup Instructions

### 1️⃣ Clone repo

```bash
git clone <your-repo-url>
cd project
2️⃣ Backend setup
cd backend
npm install
npm start
3️⃣ Frontend setup
cd frontend
npm install
npm run dev

```
### 4️⃣ Environment Variables

- Create .env in backend:

- MONGO_URI=your_mongo_url
- CLOUD_NAME=your_cloud_name
- CLOUD_API_KEY=your_api_key
- CLOUD_API_SECRET=your_api_secret
